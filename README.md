# video-player-onenote

# 🎬 Video Player para OneNote - GitHub Pages

Player de vídeo compatível com links do OneNote, hospedado no GitHub Pages.

## 🚀 Demo
https://seu-usuario.github.io/video-player-onenote

## ✨ Funcionalidades
- ✅ Carrega vídeos do YouTube, Vimeo, Dailymotion
- ✅ Interface estilo GitHub Dark
- ✅ Responsivo (celular/tablet/desktop)
- ✅ Cache local dos últimos vídeos
- ✅ Compatível com links do OneNote

## 📌 Links Suportados
- YouTube (youtube.com, youtu.be)
- Vimeo (vimeo.com)
- Dailymotion (dailymotion.com)
- TED (ted.com)
- OneNote (links copiados)

## 🔧 Como usar
1. No OneNote, copie o link do vídeo
2. Cole no player
3. Clique em "Carregar Vídeo"

## 🏗️ Deploy próprio
1. Fork este repositório
2. Ative GitHub Pages em Settings
3. Pronto!
